Class RendererMap

An object which manages a mapping of cell renderers. The namespace for the RendererMap class statics.

Notes

This class is used to configure cell renderers for a data grid.

set

  • set(region: CellRegion, metadata: Metadata, renderer: CellRenderer): void
    • Defined in renderermap.ts:108

    Set the cell renderer for a particular region and metadata.

    Parameters

    • region: CellRegion

      The cell region of interest.

    • metadata: Metadata

      The metadata to match against the model.

    • renderer: CellRenderer

      The cell renderer to set in the map.

      Notes

      The keys and values in the supplied metadata are matched against the metadata supplied by the data model. The given metadata must be an exact matching subset of the model metadata in order for there to be a match.

      Matches are ranked according the number of matched values, with ties broken based on the priorty order given to the constructor.

      Non-string metadata values are ignored.

    Returns void
