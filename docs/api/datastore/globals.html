combineReducers

  • combineReducers<S>(reducers: ReducerMap<S>): Reducer<S>
    • Defined in reducer.ts:55

    Create a single reducer from a mapping of reducers.

    Type parameters

    • S

    Parameters

    • reducers: ReducerMap<S>

      A mapping of where each key is the name of a branch in the state object, and the value is a reducer to be applied to that branch.

    Returns Reducer<S>

    A single combined reducer function.

    Notes

    This function should only be used for state which is a vanilla spreadable JS object.

    When the combined reducer is invoked, the new state is created by applying each reducer in the map to its respective branch of the state. State branches which are not included in the reducers map are copied without modification into the result state.
